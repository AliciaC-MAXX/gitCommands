Git Commands Cheatsheet

git status - shows file changes, if a file isn't tracked or needs to be added to the staging area

git commit

git commit -a -m

git pull

git push

git branch

git branch [file name]

git clone

git fetch

git merge

git checkout

git remote

git remote add

git diff


Terminal Commands

cd ~ - change directory into home
cd [file name] - change directory into chosen file
